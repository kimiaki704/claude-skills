# ジャンル別 検索クエリパターン

Phase 3 の WebSearch で使う検索クエリのパターン集。
`[期間]` は実際の日付や期間に置き換える（例: `since:2025-01` / `2025 Q1` / `last week`）。

---

## iOS / Swift / SwiftUI

```
Swift [バージョン] release notes [期間]
SwiftUI updates [期間]
Swift Package Manager changelog [期間]
Xcode [バージョン] release [期間]
iOS [バージョン] developer changes [期間]
Swift evolution accepted proposals [期間]
```

```
Swift 最新リリース [期間]
SwiftUI 新機能 [期間]
Xcode アップデート [期間]
```

**注目すべきソース**:
- https://swift.org/blog/
- Apple Developer News
- Swift Forums (forums.swift.org)
- GitHub: apple/swift releases
